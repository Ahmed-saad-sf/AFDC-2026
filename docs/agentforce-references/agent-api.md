# Agentforce Agent API

Source: https://developer.salesforce.com/docs/einstein/genai/references/agent-api?meta=Summary

## Overview

The Agentforce Agent API provides programmatic access to interact with and manage AI agents in Salesforce.

## Key Features

- Programmatic agent interaction
- Agent management and configuration
- Integration with Salesforce data and services
- Support for citations and source references
