# Agentforce Tooling Objects

Source: https://developer.salesforce.com/docs/einstein/genai/references/agents-metadata-tooling/agents-tooling.html

Manage actions and customize the planner service using the Tooling API.

## Tooling API Objects

### GenAiFunctionDefinition
Represents an action that can be added to agents.

### GenAiPlannerDefinition
Represents a planner that uses an LLM and a reasoning strategy to break down a task into smaller parts, find the best actions for each part, and invoke them.
